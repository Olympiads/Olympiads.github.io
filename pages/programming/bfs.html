BFS (Breadth-first search – Търсене в ширина)

Задача : Искаме да намерим дължините на най-краткия път от връх $v$ до останалите върхове.

За целта ще използваме обхождане на граф в ширина. Подобно на DFS ще минем през всички върхове, но тук ще искаме да пазим колко е отдалечен всеки от началния и така да обхождаме върховете.

Алгоритъм :

Използваме опашка, за да следим кой е най-близо. В началото ще имаме само началния връх $v$. Когато извадим връх $u$ от опашката ще маркираме всичките му съседи с разстояние с едно по-голямо. Така след като махнем началния на втората итерация всички върхове на разстояние $1$ ще са маркирани и вкарани в опашката. По този начин винаги ще достигнем до всеки връх по най-късия път.

Имплементация : 

vector adj[MAX_N]; /// списък на съседство
int n; /// брой върхове, MAX_N - максимален брой върхове
int starting_v; /// началния връх
queue q; /// върховете, които чакат да се обходят
bool used[MAX_N]; /// пазим дали е обхождан върха
int dist[MAX_N]; /// в dist[v] пазим разстоянието от v до началния

void bfs()
{
    q.push(starting_v); /// вкарваме началния
    used[starting_v] = true;
    dist[starting_v] = 0;
    while(!q.empty()) /// докато има необработени
    {
        int u = q.front();
        for(int i=0; i < adj[u].size(); i++)
        {
            if(used[adj[u][i]]==0) /// недостигнатите ги слагаме в опашката
            {
                used[adj[u][i]] = true;
                dist[adj[u][i]] = dist[u]+1;
                q.push(adj[u][i]);
            }
        }
    }
}
